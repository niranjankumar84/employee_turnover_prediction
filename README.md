# Employee Turnover Prediction

This project predicts whether an employee is likely to leave the company using Machine Learning.

## Features
- Job Satisfaction
- Age
- Monthly Income
- Work-Life Balance
- Training Hours

## Model
Logistic Regression with Regularization (L1 & L2)

## Evaluation
Accuracy score used for performance evaluation.

## Technologies
Python, Pandas, Scikit-learn, Jupyter Notebook
